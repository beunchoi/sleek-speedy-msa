## 프로젝트 소개
sleek speedy는 대규모 트래픽을 효율적으로 처리하여 한정 수량의 상품에 대한 선착순 구매를 지원하는 서비스입니다. 
사용자는 원하는 상품을 위시리스트에 담고 주문 및 결제를 진행하여 선착순으로 빠르게 결제를 완료하면 상품을 구매할 수 있습니다.
